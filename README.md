# Tienda de Libros
Proyecto de página web responsive para una librería ficticia.
La página fue desarrollada utilizando HTML, CSS y JavaScript, aplicando etiquetas semánticas y diseño adaptable para distintos tamaños de pantalla.

## Características
- Landing page interactiva
- Navegación dinámica entre secciones
- Diseño responsive
- Uso de Flexbox y CSS Grid
- Sección de productos organizada en cards
- Sección de reseñas con Grid Layout
- Formulario de contacto funcional con Formspree
- Uso de Google Fonts e Iconify

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Formspree
- Google Fonts
- Iconify
